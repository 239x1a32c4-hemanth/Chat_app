Flask Chat Application
  A secure real-time chat application built using Flask.
  This project implements user authentication with JWT, password hashing with Bcrypt, and message storage using SQLite and SQLAlchemy.
  Users can register, log in, and send messages securely through protected API routes.
  Flask-Based Secure Chat Application

This project is a full-stack chat application developed using Flask. It demonstrates secure authentication, RESTful API design, and database integration.
